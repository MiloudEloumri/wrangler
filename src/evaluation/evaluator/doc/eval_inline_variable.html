Description

This module simplifies variables definitions. When a variable is created and used after, the varible evaluates to its value. Only variables defined via a match expression of the format: VarName = Expr can be inlined. Consider the function below:
f() -> X = 1,
X.
If the user chooses to simplify the function f/0, the Wrangler tool will identify that there is a variable declaration (X=1) and the variable X in the second clause will be simplified to 1.
