Description

If Core - Where possible, simplifies if expressions by the result of their evaluation.

Examples of usage:
  • if
    false -> [2];
    true -> []
    end.
     is simplified to [].
  • begin
    X = true,
    if
    X -> [2];
    true -> []
    end
    end.
     becomes
    begin
    X = true,
    [2]
    end.
