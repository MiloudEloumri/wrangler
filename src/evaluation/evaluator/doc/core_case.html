Description

Case Core - Where possible, simplifies case expressions by the result of their evaluation.

Examples of usage:
  • case [1,2] of
    [H | T] -> ok;
    _ -> error
    end.
     is simplified to ok.
  • begin
    X = true,
    case X of
    true -> first;
    _ -> second
    end
    end.
     becomes
    begin
    X = true,
    first
    end.
